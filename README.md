## 快速开始

### 下载安装

1. 下载 `WenQuDownloader_v3.0.0.zip`
   - [点击下载](https://github.com/liusige377/wenqu-dl/releases/download/v3.0.0/WenQuDownloader_v3.0.0.zip)
   - 或访问 [下载页面](https://liusige377.github.io/wenqu-dl/)
2. 解压 ZIP 到任意目录（如 `D:\问渠下载器`）
3. **右键 → 以管理员身份运行** `install.bat`
4. 安装完成后浏览器扩展（见下方）
5. 打开管理界面：http://127.0.0.1:15889

### 安装浏览器扩展

1. 打开 Chrome / Edge
2. 地址栏输入：`chrome://extensions`（Edge用 `edge://extensions`）
3. 打开右上角 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」**
5. 选择 `问渠下载器\resources\extension` 文件夹
6. 完成！浏览器下载自动被问渠接管

> 扩展会自动检测网页视频（m3u8/HLS），右键菜单可嗅探下载。

### 卸载

双击运行 `卸载.bat`（在安装目录内）

## 技术原理

问渠下载器基于 Node.js 多线程分片下载，核心原理：

- **32线程并发**：同时从服务器请求文件不同区块（HTTP Range）
- **断点续传**：每个分片独立保存，下载中断后重启自动衔接
- **智能队列**：多个任务自动排队，避免带宽挤占
- **自动降线程**：检测到某分片持续超时时自动减少并发，保证稳定性
- **视频嗅探**：深度Hook浏览器fetch/XHR，自动检测页面中的m3u8流和视频文件

## 常见问题

**Q: 安装后浏览器扩展无法拦截下载？**  
A: 确认已启用开发者模式，且扩展加载的是 `resources\extension` 目录（不是根目录）

**Q: 下载卡在 99% 不动？**  
A: 某些服务器对单IP有并发限制，下载器会自动降线程，稍等即可完成

**Q: 想改下载线程数？**  
A: 管理界面 → 设置 → 默认线程数

**Q: 如何更新到新版本？**  
A: 重新下载新版本 ZIP，解压后覆盖原文件，重新运行 install.bat（用户数据保留在 %APPDATA%）
