安装步骤
1. 打开 Chrome / Edge 浏览器
2. 地址栏输入：chrome://extensions（Edge用 edge://extensions）
3. 打开右上角 「开发者模式」 开关
4. 点击 「加载已解压的扩展程序」
5. 选择问渠下载器安装目录下的 resources\extension 文件夹
